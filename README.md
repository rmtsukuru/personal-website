# Personal Website
This is my personal/professional website. It's a very simple Node server used mostly to serve static HTML content, but I added EJS for templating (read: layout) support and some data (such as current year) is generated dynamically on the backend.

# Installation
After cloning the repository, run `npm i` then `npm start` (or `npm run dev` if you want the server to automatically restart on code changes). Then visit `http://localhost:8000` in your browser to view the site running locally.
